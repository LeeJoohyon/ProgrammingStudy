# 현재위치
터미널 창의 현재 폴더 위치를 나타 냅니다.
```
$ pwd
/Users/joohyun/Projects/computer-sc/ProgrammingStudy

```

# 디렉토리 이동 (change-directory)
```
$ cd [이동할 폴더]
```

# 폴더 생성 (make directory)
```
$ mkdir [생성할 폴더이름]
```

# 폴더 삭제 (rm)

```
$ rm [폴더]
```

# 복사
```
$ cp [폴더]
```

# 현재 디렉토리 안에 파일들 목록을 보여준다.
```
$ ls -al
```

# 선택한 폴더 나 파일 이동

```
$ mv mv web /bin "web"
```
